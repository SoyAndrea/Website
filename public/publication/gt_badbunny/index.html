R-Ladies Buenos Aires Meetup: Creando tablas en {gt} con Spotify API & Bad Bunny

En colaboración con Kari Bartolomé desarrollamos el taller mostrando cómo podemos obtener información desde Spotify API y plasmarla en una tabla usando el paquete {gt} que permite mostrar mucho más que solo números. Aunque el paquete no es parte del Tydiverse, su sintaxis y estructura es compatible con esta poderosa herramienta para el análisis de datos. Y cómo si fuera poco, nos traen un ejemplo con info de nada más y nada menos que Bad Bunny 🐰🎵
